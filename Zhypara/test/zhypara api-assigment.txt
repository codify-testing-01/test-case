GET 
Сценарий : получение заказа по id
Пред условия
Зайти на сайт: https://petstore.swagger.io/v2/store/order/5
order_id = 5

Шаги:
1. В Postman  делаем запрос на URL из пред условий (1),
где order_id из предусловий(2)

Ожидаемый резултат;
код ответа:200
тело ответа
{

  "petId": 5,
  "quantity": 3,
  "shipDate": "2022-12-13T07:36:26.695+0000",
  "status": "placed",
  "complete": true
}
